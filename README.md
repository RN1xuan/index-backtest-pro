# 📈 指数多模式策略量化回测系统 (Index Strategy Backtest Pro)

> 🚀 一个**纯前端、零配置、开箱即用**的 A 股 / 宽基指数网格与定投策略可视化回测工具。
> 您的所有金融数据仅在本地浏览器解析，绝不上传任何服务器，100% 保护您的持仓与策略隐私。

---

## ✨ 核心功能亮点

1. **🛡️ 纯本地计算**：基于 HTML5 + JS 编写，双击 `index.html` 即可运行，完全离线可用。
2. **🎛️ 四大交易核心模式**：
   * **📉 左侧抄底**：越跌越买，设定跌幅阈值自动接飞刀，积攒低价筹码。
   * **📈 右侧确认**：先跌后涨双重确认，拒绝半山腰，确认底部反转再动手。
   * **🌊 波段区间**：设定点位区间，低位全仓杀入，高位清仓逃顶，复利滚存。
   * **⏳ 定期定额**：无视牛熊涨跌，设定交易日间隔无脑定投（DCA）。
3. **💰 真实复利与滚动止盈**：支持自定义“触发收益率”与“清仓比例”。止盈后的资金会进入“闲置现金池”，并在下次买入时优先使用，实现**用利润去投资**的真实复利复盘。
4. **📊 极客级可视化大屏**：内置 4 大联动图表（原始走势、绝对盈亏、基准收益率对比、总资产曲线），买入点🔴与清仓点🟢全景追踪。

## 📥 数据获取指南 (非常重要)

本系统需要您提供指数的**历史日线数据**才能运行。您可以免费从官方获取权威数据：

1. **访问中证指数官网**：🔗 [https://www.csindex.com.cn/zh-CN/indices/index#/indices/family](https://www.csindex.com.cn/zh-CN/indices/index#/indices/family)
2. **搜索目标指数**：在页面搜索框输入您想回测的指数（例如：`沪深300`、`中证500` 或相应的代码 `000300`）。
3. **下载历史数据**：进入指数详情页后，找到**“历史行情”**或**“数据下载”**区域，导出 `.xlsx` 或 `.csv` 格式的文件。

**⚠️ 数据格式要求：**
下载后的表格中，必须包含以下三列（系统能自动识别各类常见中英文表头）：
* **日期** (例如: `2024-01-01` 或 `20240101`)
* **收盘价** / **收盘点位** (例如: `3500.5`)
* **涨跌幅** (例如: `-3.5` 或 `1.2`)

## 🚀 快速开始

1. **克隆/下载本项目**：点击本页面右上角的 `Code` -> `Download ZIP`。
2. **解压缩**：将下载的 ZIP 文件解压到您的电脑。
3. **运行系统**：使用 Chrome、Edge 或 Safari 等现代浏览器，直接双击打开 `index.html` 文件。
4. **上传数据**：在界面左上角点击上传您刚刚在中证官网下载的 `.xlsx` 或 `.csv` 文件。
5. **自由回测**：选择您的交易模式，调整跌幅阈值、资金分配和止盈比例，点击【开始回测】，即刻生成您的财富曲线！

## 💡 策略启示录

为什么要做这个工具？因为**直觉往往是错的**。

* 以为“逢大跌就买”一定赚钱？在单边阴跌的熊市，本金可能早早打光。
* 以为“每日定投”天下无敌？在单边暴涨的新能源赛道，定投反而严重拖累了收益率，跑不过一把梭哈。
* 以为“止盈”绝对正确？卖飞主升浪的痛，只有看过复盘图表上的绿点才知道。

**没有完美的策略，只有匹配特定行情的策略。** 愿这个小工具能帮助您在复盘中看清市场的真相，找到最适合您性格与资金体量的交易之道。
