# Index Backtest Pro

这是一个基于原生前端技术（HTML/CSS/JS）实现的指数策略回测工具。项目无需部署后端服务或数据库，可直接在浏览器中运行，通过 API 接入获取历史行情数据，并支持多种基础定投及趋势交易策略的回测与可视化分析。

## 功能特性

* **纯前端无服务架构**：本地双击 `index.html` 即可运行，计算过程及数据均在前端完成。
* **在线数据接入**：通过 JSONP 跨域请求集成东方财富 API，支持 A 股、港股、美股及各类 ETF 标的的代码及拼音模糊搜索，并动态拉取历史日 K 线数据。
* **多策略支持**：
    * **左侧交易**：设定单日跌幅阈值，触发条件即执行固定金额或阶梯金额买入。
    * **右侧交易**：设定前置下跌条件与后置反弹突破阈值，确认底部信号后买入。
    * **定期定额 (DCA)**：按固定的交易日间隔执行无差别定投。
    * **波段交易**：设定具体的价格区间（箱体），跌破低点全仓买入，突破高点全部清仓。
* **动态止盈机制**：可独立开启按比例止盈功能，设定目标收益率及触发后的卖出仓位比例。
* **数据可视化**：基于 Chart.js 渲染回测结果，包含标的价格与买卖点标注、策略收益率与指数涨跌幅对比、总资产与累计投入本金走势，并提供完整的交易流水明细。

## 使用说明

### 1. 运行环境
本项目对环境无特殊要求。
* **常规运行**：下载代码后，直接使用 Chrome、Edge 等现代浏览器打开 `index.html`。
* **推荐运行**：为避免个别浏览器对 `file://` 协议的安全限制，推荐使用 VS Code 插件 `Live Server` 或通过 Python 简易 HTTP 服务启动（例如 `python -m http.server`）。

### 2. 操作流程
1.  **标的检索**：在顶部输入框输入标的名称、代码或拼音缩写（如“沪深300”或“000300”）。
2.  **选择标的**：必须在弹出的下拉提示列表中点击选中目标，系统会自动绑定对应的内部标的代码。
3.  **获取数据**：点击“拉取”按钮，等待系统加载历史日线数据。
4.  **配置参数**：在面板中选择回测时间范围、交易模式及对应的资金与点位参数。
5.  **执行回测**：点击“开始回测”，页面下方将生成统计面板、走势图表及交易记录。

## 技术栈

* **前端结构**：HTML5 + CSS3 (Grid 响应式布局)
* **逻辑实现**：原生 JavaScript (ES6+)
* **图表库**：[Chart.js](https://www.chartjs.org/) (CDN 引入)
* **数据接口**：东方财富 Search API & K-Line API
